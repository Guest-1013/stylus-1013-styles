# 简介

本仓库是我为一些网站编写的自定义CSS文件，适用于[Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)这一chrome插件。

# 使用方法

## 手动导入

1. 安装Stylus后，在想要应用样式的网站上点击插件图标，进入插件窗口后点击域名，进入代码编辑窗口；
2. 在代码编辑窗口，粘贴对应网站的代码；
3. **（重要）** 在代码编辑器上方有一个指定域名和匹配方式的输入条；在左侧选择“域名”（而非“URL”，“前缀”等；虽然可能也能用），并检查其右侧的域名是否为想要生效的域名；
4. 此时自定义样式应该已经生效了。
