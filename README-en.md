# Introduction

This repository contains custom CSS files I've written for various websites, designed for the [Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) Chrome extension.

# Usage

## Manual Import

1. After installing Stylus, click the extension icon on the website you want to style, then click the domain name in the popup to enter the code editor;
2. In the code editor, paste the CSS code for the corresponding website;
3. **(Important)** At the top of the code editor there is a bar for specifying the domain and matching mode; on the left, select "Domain" (rather than "URL", "Prefix", etc. — though those may also work), and verify that the domain on the right is the one where you want the styles to take effect;
4. The custom styles should now be active.
